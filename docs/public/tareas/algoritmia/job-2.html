2. Codigo de la pagina 9 del manual extenso

Agosto 9.2022 | Guillermo Mestas

Extension:

mensaje4 = 'hola' + ' ' + 'mundo'
print(len(mensaje4))

Salida:

> 10

Encontrar:

mensaje5 = 'Hola Mundo'
mensaje5a = mensaje5.find("Mundo")
print(mensaje5a)

Salida:

> 5

Si la sub-cadena no está presente el programa imprimirá el valor -1.

mensaje6 = 'Hola Mundo'
mensaje6a = mensaje6.find("ardilla")
print(mensaje6a)

Salida:

> -1

Minusculas:

mensaje7 = "HOLA MUNDO"
mensaje7a = mensaje7.lower()
print(mensaje7a)

Salida:

> hola mundo

Reemplazar:

mensaje8 = "HOLA MUNDO"
mensaje8a = mensaje8.replace("L", "pizza")
print(mensaje8a)

Salida:

> HOpizzaA MUNDO

Cortar:

mensaje9 = "Hola Mundo"
mensaje9a = mensaje9[1:8]
print(mensaje9a)

Salida:

> ola Mun

Puedes sustituir las variables por números enteros como en este ejemplo

mensaje9 = "Hola Mundo"
startLoc = 2
endLoc = 8
mensaje9b = mensaje9[startLoc: endLoc]
print(mensaje9b)

Salida:

> la Mun
